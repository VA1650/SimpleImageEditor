# SimpleImageEditor (Dark Blue Edition)

Растрово-векторный графический редактор, реализованный на базе стандартной графической библиотеки `tkinter` (с кастомной стилизацией через `ttk`) и движка обработки изображений `Pillow (PIL)`. 

Интерфейс спроектирован по модульной сетке с адаптивным изменением размеров холста, поддержкой многослойной логики при выделении объектов и сквозными глобальными хоткеями.

## 🛠 Зависимости и развертывание

Для корректной работы всех модулей (включая фильтры и динамический парсинг системных шрифтов) требуются сторонние пакеты `Pillow` и `matplotlib`.

### Установка пакетов
```bash
pip install Pillow matplotlib
```

### Запуск приложения
```bash

python image_editor.py
```

## 🏗 Архитектурные особенности и алгоритмы

* **Оптимизированный алгоритм заливки (Flood Fill):** Вместо стандартного рекурсивного метода, который при больших разрешениях неизбежно выбрасывает `RecursionError` (переполнение стека вызовов), в коде развернута итеративная заливка на базе двусторонней очереди (`collections.deque`). Алгоритм работает по принципу обхода в ширину (BFS) с жестким лимитом итераций для защиты от зависания интерфейса.
* **Неразрушающий стек истории (Undo/Redo):** Реализована глубокая история изменений без деградации производительности. Состояние холста кэшируется в виде иммутабельных копий объектов `PIL.Image.copy()`. Ограничитель глубины истории (`MAX_HISTORY = 30`) предотвращает утечки и избыточное потребление оперативной памяти при длительных сессиях.
* **Изолированный слой выделения (Selection Layer):** При активации инструмента выделения (`select_rect`) выбранная область программно вырезается (`crop`), исходный слой под ней заполняется фоновым цветом, а сам фрагмент проецируется на холст как независимый объект с собственными координатами. Слияние слоев (фиксация) происходит автоматически при смене инструмента, создании нового выделения или двойном клике.
* **Динамическая типографика:** Компонент работы с текстом не использует хардкод путей. С помощью `matplotlib.font_manager` при старте приложения сканируются системные директории ОС, вытягиваются реальные имена шрифтов и маппятся с их физическими `.ttf` / `.otf` файлами.

## ⌨️ Управление и горячие клавиши

Все ключевые триггеры привязаны через `bind_all` к контексту главного окна, что позволяет мгновенно менять инструменты без отрыва рук от клавиатуры.

### Системные операции
* `Ctrl + N` — Создать новый чистый холст (800x600px).
* `Ctrl + O` — Импортировать изображение (`PNG`, `JPG`/`JPEG`, `BMP`).
* `Ctrl + S` — Сохранить текущее состояние холста в файл.

### Быстрый выбор инструментов
* `Ctrl + 1` — Карандаш
* `Ctrl + 2` — Прямая линия
* `Ctrl + 3` — Прямоугольник
* `Ctrl + 4` — Овал
* `Ctrl + 5` — Заливка области
* `Ctrl + 6` — Прямоугольное выделение
* `Ctrl + 7` — Вставка текста
* `Ctrl + 8` — Пипетка (взятие цвета с холста)
* `Ctrl + 9` — Ластик

### Правка и трансформация
* `Ctrl + Z` — Шаг назад (Отмена действия)
* `Ctrl + Y` — Шаг вперед (Повтор действия)
* `Ctrl + Enter` — Обрезать холст по границам активного выделения
* `Двойной клик` — Снять выделение с фиксацией изменений на холсте
